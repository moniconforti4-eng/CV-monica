Currículum Vitae Interactivo – Mónica Conforti
Proyecto desarrollado como parte de la evaluación final del Módulo 2 – Desarrollo Front-End del programa Kibernum IT Academy. Consiste en la creación de un CV interactivo, responsivo y publicado mediante GitHub Pages, utilizando tecnologías modernas de desarrollo web.

Características principales
Diseño responsivo utilizando Bootstrap 5

Navbar funcional para navegar entre secciones

Botón para alternar entre modo claro y oscuro

Interactividad implementada con JavaScript 

Formulario de contacto con validación

Animaciones y efectos visuales

Código organizado en archivos separados (HTML, CSS, JS)

Proyecto desplegado en GitHub Pages
